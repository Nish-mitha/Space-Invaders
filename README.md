# Space-Invaders

### Space Invaders is 2D game made using Python and Pygame.

**Play the Game in your system by following the below steps**

1. `Fork and clone the project`
2. `Install Pygame in your system`

       pip install pygame
3. `Run the Space-Invaders Game`

       python3 main.py
